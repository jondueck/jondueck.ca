What is ARIA?

ARIA (Accessabile Rich Internet Applications) is a set of of attributes applied to HTML attributes. ARIA assists in telling browsers what the role or state of an attribute is. This is particularly valuable in situations where accesability is key. ARIA can aid screen readers.

ARIA Attribute What?
role="main" The primary content of the page
role="banner" A region that contains mostly site-oriented content, rather than page-specific content.
role="navigation" A collection of navigational elements
