Creating Custom Post Types in Wordpress

  • Wordpress

When using Wordpress for a site of anything other than a blog, it’s common to need to create a custom post type.

add_action( 'init', 'create_posttype' );
    function create_posttype() {
      register_post_type( 'bulletinboard',
        array(
          'labels' => array(
            'name' => __( 'Bulletins' ),
            'menu_name' => __( 'Bulletin Board' ),
            'singular_name' => __( 'Bulletin' ),
            'view_item' => __( 'View Bulletin' ),
            'add_new' => __( 'Add New Bulletin' ),
            'add_new_item' => __( 'Add New Bulletin' ),
          ),
          'public' => true,
          'has_archive' => true,
          'show_in_nav_menus' => TRUE,
          'rewrite' => array('slug' => 'bulletinboard'),
        )
      );
    }
